# Java Logic & Utility Programs ☕

This repository is a collection of fundamental Java programs focused on building strong algorithmic logic and problem-solving skills. All programs are organized within the `mathutility` package.

## 📂 Project Contents (`mathutility`)

This folder contains the following logic implementations:

* **MainDoc.java**: The main entry point that integrates and executes all utility classes.
* **GCD & LCM**: Methods to calculate the Greatest Common Divisor and Least Common Multiple.
* **Prime Number**: Logic to determine if a number is prime.
* **Reverse & Palindrome**: Programs to flip digits and check for palindromic numbers.
* **Pascal’s Triangle**: Generates a triangle of binomial coefficients.
* **Floyd’s Triangle**: A nested loop implementation of a natural number triangle.

## 🛠️ Technical Stack
* **Language:** Java
* **IDE:** VS Code
* **Version Control:** Git & GitHub
